================================================================================
                        🆘 TROUBLESHOOTING QUICK REF
================================================================================

Problem: Docker won't start
→ Solution: Make sure Docker Desktop is installed and running

Problem: Port already in use (8000, 8001, 8002)
→ Solution: Run "netstat -ano | findstr :8000" then "taskkill /PID <pid> /F"

Problem: Flutter not found
→ Solution: Install Flutter from https://docs.flutter.dev/get-started/install

Problem: No devices found
→ Solution: Run "flutter devices" to check. Start emulator or connect phone.

Problem: Camera permission denied
→ Solution: Settings → Apps → RepSense → Permissions → Enable Camera

Problem: Can't connect to backend from phone
→ Solution: Update mobile/.env with your computer's IP address (not localhost)

For detailed troubleshooting, see: QUICK_START.md or SETUP_COMPLETE.md
